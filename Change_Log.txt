Offshore SST Map — Change Log

Version History

v0.1 — Initial Build (Early August 2025)
	•	Created Streamlit app to visualize daily SST (Sea Surface Temperature) for NJ→MA AOI using NOAA CoastWatch ERDDAP.
	•	Implemented:
	•	Dataset search via ERDDAP API.
	•	Automatic dataset selection with preference for MUR → GHRSST → OISST.
	•	SST conversion from Kelvin to Fahrenheit.
	•	Heatmap rendering with Folium ImageOverlay.
	•	AOI polygon display via GeoJson.
	•	Early Issues:
	•	Hover-over SST reading not implemented (relied on click events).
	•	Map recentered on every click/interaction.
	•	Heatmap overlay alignment inconsistent with coastline.
	•	Visual appearance: tiles bled over land in some areas.

⸻

v0.2 — First Hover Attempt
	•	Added tooltip system using Folium click events to show SST values.
	•	Added MousePosition plugin to show lat/lon in real time.
	•	Problems:
	•	Tooltips only appeared after clicking.
	•	Map jumped/recentered after every click.
	•	Heatmap alignment still imperfect.

⸻

v0.3 — Hover-over Conversion & Initial Stability Fixes
	•	Replaced click events with first implementation of hover-over SST using st_folium last_mouse_position.
	•	Added persistent view_center and view_zoom in st.session_state to try to reduce jumping.
	•	Issues:
	•	Hover values inconsistent, especially when zoomed in.
	•	Map still reloaded and reset view on many interactions.
	•	Hover logic tied to Streamlit updates → caused flicker.

⸻

v0.4 — Leaflet-Native Tooltips
	•	Switched hover-over from Streamlit mouse tracking to Leaflet-native tooltips:
	•	Added folium.CircleMarker at grid cell centers with invisible styling.
	•	Each marker has a tooltip with precomputed °F value.
	•	Added tooltip density mode:
	•	Sparse (fastest), Normal, Dense (more coverage).
	•	Results:
	•	Tooltips far more reliable.
	•	Still occasional gaps between markers, especially when zoomed in.

⸻

v0.5 — Coastline Masking & Visual Enhancements
	•	Added AOI-based masking to hide SST values outside polygon.
	•	Added land overlay layer so heatmap is visually beneath coastlines.
	•	Improved heatmap alignment by correcting raster orientation (row 0 = North, col 0 = West).
	•	Added upsampling (1x, 2x, 3x) for smoother visuals.
	•	Improved color scale handling:
	•	Fixed scale: 48–90°F.
	•	Adaptive scale: 5th–95th percentile of AOI data.
	•	UI updates:
	•	Sidebar controls for upsample factor, scale lock, tooltip density, and lock/unlock map view.

⸻

v0.6 — Map Stability & ERDDAP Enhancements
	•	Locked view option now prevents any recentering after user pans/zooms.
	•	Map no longer jumps to AOI bounds after every interaction.
	•	ERDDAP search logic improved:
	•	Uses primary and fallback search terms from config.json.
	•	Scoring system prioritizes daily SST datasets, MUR highest.
	•	Variable detection improved from .das file to find SST variable name.
	•	Grid spacing (km) calculated from dataset metadata.

⸻

v0.7 — Major Update (2025-08-11)

Hover-over SST Fix
	•	Leaflet-native tooltips fully implemented; no reliance on click events.
	•	Tooltip density modes tuned for better coverage at different zoom levels.
	•	Fixed issue where tooltips disappeared when zoomed in by recalculating stride based on grid resolution.

Map Stability Improvements
	•	view_center and view_zoom persist via st.session_state.
	•	“Lock map view” prevents any recenter after interaction.

Coastline Masking
	•	SST raster clipped to AOI.
	•	Added land overlay layer so coastlines mask heatmap for clean fit.
	•	Improved tile alignment with coastline curves.

Color & Scale
	•	Added fixed color scale option (48–90°F).
	•	Improved adaptive scaling to prevent extreme outliers from skewing.
	•	Added 1x/2x/3x upsample for smoother rendering.

Backend/Data
	•	Robust Kelvin/Celsius/Fahrenheit handling in SST conversion.
	•	ERDDAP fetch process more fault-tolerant and better dataset selection.

Known Issues / Next Steps:
	•	Tooltip coverage still a performance trade-off at very high zooms.
	•	Coastal masking limited by dataset resolution; higher-res shoreline masks could improve fit.
	•	Future possibility: vector-based mask for pixel-perfect coastline clipping.

⸻
